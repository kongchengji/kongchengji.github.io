SQL数据库学习之路（三）
发表于 2018-10-20| 更新于 2020-05-10|SQL数据库
阅读量:

一.表数据的操作

1.简单查询 select * from 表名

2.增加数据 insert into 表名(列名) values（值）

----要求值的列名与值要位置对应

----如果所有都插入值，可以省略列名部分

insert TextInfo(Name,pwd) values('黑子','ba65d6181092fdb8e9dce84de9ff18ba') --为TextInfo表中name列和pwd列赋值

--一次性赋值多个数据
select * from ClassId
insert into ClassName
values('青龙'),('白虎'),('朱雀'),('玄武')

3.修改数据： update 表名 set 列名1=值1，列名2=值2....

update gg set [name] ='大海'

4.删除数据：delete from 表名 where ...

清空：truncate from 表名

--删除数据
delete from gg where Id>8


--清空数据
truncate table gg

5.常用辅助命令及快捷键

set statistics time on/off----在消息栏显示详细执行时间

ctrl+e ---执行

ctrl+r ----隐藏消息栏

ctrl+l -----计划任务，对SQL语句进行分析
