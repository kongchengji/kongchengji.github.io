窗口界面介绍：

scene:

在场景编译窗口中可以鼠标右键选中之后移动鼠标就可以自由的浏览场景，并选择物体对其可以进行操作，按下Q\W\E\R四个键时可以画面中的将会进行变换，第一个图标可以进行拖动、第二个图标可以对物体进行位移，第三个图标可以对物体进行旋转，第四个图标可以对物体进行放大缩小。查看整合场景时，按A\D可以进行左右移动查看，按Q\E可以进行上下移动，按W\S可以对物体进行前后移动。

Game :

玩家将要看到的画面，单击中的左侧播放按钮就可以进行运行。中间是暂停按钮，右侧是播放下一帧按钮。在游戏窗口中的Free Aspect中可以调节游戏显示的长宽。Free Aspect代表全屏显示，Standalone代表所要发布的版本。右侧的Maximize on Play代表下一次游戏播放时将全屏播放。Stats控制是否显示游戏运行时的各项参数。Gizmos控制是否显示游戏的各种标记。

Hierarchy:

所有场景内的物体都以列表展示在Hierarchy窗口中，可以利用搜索菜单搜索场景内的物体，所有非搜索物体都会以灰色呈现。三角符号代表父子关系。

Project:

所有项目资源都可以通过project项目资源窗口查看，它与文件夹显示的内容是一致的。favorites菜单可以用来查找特定类型的资源，也可以自己定义要的资源。

Inspector:

所有项目资源都可以在其中进行编辑，不同属性的资源会有不同属性的编辑内容。

其他窗口：

layers表示在scene窗口中可视的层。例如选取场景中的一个物体，改变它的图层。右侧的下拉窗口表示scene窗口显示的模式。

菜单界面

File文件菜单

file菜单可以新建场景、打开场景；保存场景；新建项目、打开项目、保存项目；设置项目、发布项目；退出unity。

Edit编辑菜单

edit菜单主要负责一些编辑的命令。比如：暗度、灰度、剪切、复制等。

Preferences：对unity本身的基础设置菜单。在skin(皮肤)中可以修改窗口的基础色调。--仅专业版使用

点击colors--playmode tint修改颜色，会在选择运行时改变界面色差。

project settings:

input代表游戏的输入管理

tags and layers:可以控制游戏的层级，以及不同标识的tag

audio:游戏的声音

time：游戏刷新的时间等。

player:从发布版本中可以找到的内容

windows窗口菜单

Animation

动画绘制：可以对animation进行修改，但这样做非常复杂，所有大部分的动画制作都会交给美工使用3dmax进行操作。

Profiler

显示关于CPU和GPU的使用情况，可以看到游戏所消耗的资源。

Version control

用来控制游戏版本，由于兼容性不足，平时很少使用。

Animator

使用动画状态机的方式将不同的动画融合在整个体系当中，
