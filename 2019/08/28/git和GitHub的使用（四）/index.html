GitHub的使用练习

本地库文件创建

先在git中创建一个本地库文件gay.txt，将其添加到本地库中。

创建GitHub远程库

在GitHub右上角可以对仓库进行创建

创建GitHub仓库的界面

创建好的库

将本地库的内容通过push上传到远程库

打开GitHub仓库

我们可以在本地库中保存远程库的地址

存储远程库地址命令：git remote add origin

查看远程库地址命令：git remote -v

fetch是地址取回 push是地址推送

推送操作

推送命令：git push origin master

会跳出弹窗去添加GitHub的账号和密码

完成之后会显示

刷新GitHub仓库，可以看到之前本地库文件已经上传到远程库了

克隆操作

命令：git clone 远程库的地址

效果：

  • 完整的把远程库下载到本地
  • 创建origin远程地址别名
  • 初始化本地库

一起学习，一起进步 -.- ，如有错误，可以发评论
