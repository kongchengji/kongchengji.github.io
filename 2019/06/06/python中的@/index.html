python中的@
发表于 2019-06-06| 更新于 2020-05-10|python
阅读量:

@是python中的一个修饰符

官方文档

@dec2  
    @dec1  
    def func(arg1, arg2, ...):  
        pass

等价于

def func(arg1, arg2, ...):  
    pass  
func = dec2(dec1(func))

具体示例：

def a(b):
    print("************")
    return b

@a
def c():
    print("aaaaaaaaaaaa")

c()
