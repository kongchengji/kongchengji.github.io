创建第一个python项目：

点击桌面的pyCharm图标

进入pyCharm应用界面

右键新建一个python程序

在python中print输入一段语句

点击run运行按钮，在console界面就会出现自己输入的语句

python程序文件是以.py结尾的。

在命令提示符cmd中输入dir能显示当前目录下所有的文件名

在命令提示符cmd中输入python first.py，相当于运行first.py文件。

cd .. 回到上一级目录

注释方式：

python注释为 #注释单行

'''注释内容 ''' 注释多行

"""注释内容"""

参考：http://www.runoob.com/python3/python3-comment.html

输出方式：

Python两种输出值的方式: 表达式语句和 print() 函数。

print是打印到屏幕一些信息

可以使用" , "分隔让一句print打印多个字符串，字符串之间打印时遇到逗号会加空格

print("this is my first python","ok I is doing test","nothing to do")

print中还可以直接进行简单的运算，运行结果将会变成19

print(10+5+3+1)

输入方式：

input输入函数，获取外部变量的值

Python提供了 input() 内置函数从标准输入读入一行文本，默认的标准输入是键盘。

input 可以接收一个Python表达式作为输入，并将运算结果返回。可以在input中写字符串，将会在运行后显示在输入之前

goal = input("请输入： ")
print("the goal is", goal)

python数据类型：

  1. number数字类型：1.整数 2.浮点数（小数） 3.复数
  2. 字符串string
  3. Boolean布尔值：true 和 false
  4. none空值
  5. list列表
  6. tuple元组
  7. dict字典
  8. set集合
  9. 没有数组

查看python中关键字：

import keyword
print( keyword.kwlist )
